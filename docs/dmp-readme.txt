TOTO JE README SOUBOR PRO MAUTIRTNÍ PRÁCI
------------------------------------------
K instalaci aplikace budete potřebovat prostředí XAMPP.
V dokumentaci je popsáno, jak získat prostředí, i jak pracovat
s aplikací jako takovou.

Co se týče E-mail systému, v aplikaci máte předpřipravený soubor
".env.example". Tento soubor zkopírujte, přejmenujte ho na .env,
a doplňte údaje svého SMTP serveru / e-mailového ověření.
Moje nastavení bylo takové, že e-maily se odesílají z mé
osobní gmailové adresy.

Též jsem v kořenovém adresáři nechal soubor setup.php, ve kterém
je toto popsané trochu detailněji
